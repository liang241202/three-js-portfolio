---

## Setup on a new machine

Copy these files to Claude Code's project memory directory:

```
~/.claude/projects/-Users-<username>-Three-js-projects/memory/
```

Or run from repo root:
```bash
mkdir -p ~/.claude/projects/-Users-$(whoami)-Three-js-projects/memory
cp .claude/memory/* ~/.claude/projects/-Users-$(whoami)-Three-js-projects/memory/
```
